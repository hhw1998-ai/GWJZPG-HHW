# DESIGN.md

## 产品定位
通用型岗位价值评估 SaaS 工作台。面向 HR 专业人士和企业管理层，提供从数据导入、在线评分到排名分析的一站式岗位价值评估工具。

## 气质与意象
- **关键词**：专业、温润、信赖、精准
- **意象场景**：午后阳光透过百叶窗洒在深色胡桃木办公桌上，一本翻开的精装评估报告旁放着一杯刚沏的乌龙茶。空气中是纸张和木质家具的淡淡香气。这不是冰冷的科技产品，而是一件被 HR 专家信赖的精密仪器。
- **差异化**：不走"科技蓝+白底+圆角卡片"的 SaaS 套路。用暖灰+琥珀金作为主调，营造沉稳、可信赖的专业感。

## Design Tokens

### 色彩
| Token | 值 | 用途 |
|-------|-----|------|
| 主色 | `#3D3630`（深胡桃棕） | 标题、主按钮、强调文字 |
| 强调色 | `#C8956C`（琥珀金） | 关键操作按钮、高亮、进度指示 |
| 背景 | `#FAF8F5`（暖白） | 页面底色 |
| 卡片 | `#FFFFFF` | 卡片、面板背景 |
| 边框 | `#E8E3DD`（暖灰边框） | 分割线、输入框边框 |
| 文字主 | `#2C2825` | 正文 |
| 文字辅 | `#8B8580` | 辅助说明 |
| 成功 | `#5B8C5A` | 完成状态 |
| 警告 | `#D4954B` | 警告提示 |

### 字体
- 标题字体：`DM Serif Display`（Google Fonts，用于 Hero 大标题和页面主标题）
- 正文字体：`PingFang SC`, `Hiragino Sans GB`, `Microsoft YaHei`（中文优先）
- 数字/代码：`JetBrains Mono`（评分数字、统计数据）

### 间距与圆角
- 页面边距：`max-w-7xl mx-auto px-6`
- 卡片圆角：`rounded-xl`（12px）
- 按钮圆角：`rounded-lg`（8px）
- 区块间距：`py-20`（大区块）、`py-12`（小区块）

### 动效
- 性格：克制、精准、有重量感
- 缓动：`cubic-bezier(0.4, 0, 0.2, 1)`（标准缓出）
- 页面进入：fade-in + 微上移（`translateY(8px) → 0`），stagger 50ms
- 悬停：`scale(1.02)` + 阴影加深，过渡 200ms
- 数字变化：CSS counter 动画

## 布局与响应式
- 桌面端优先，最大宽度 1280px
- Landing Page：单栏垂直滚动，Hero → 功能介绍 → 使用流程 → CTA
- 评估工作台：全宽表格，粘性表头+首列
- 后台管理：侧边栏（桌面端）/ 顶部 Tab（移动端）

## 设计禁忌
- 不要使用科技蓝+紫色渐变
- 不要使用纯黑（#000）文字
- 不要过度使用阴影（最多 2 层）
- 不要使用 emoji 作为图标
- 不要使用 Inter/Roboto 作为标题字体
