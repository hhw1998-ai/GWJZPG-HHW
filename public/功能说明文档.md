## 二、评估模型

### 2.1 六因素十四维度

系统采用国际通用的岗位价值评估模型，包含 **6大因素、14个评分维度**：

| 因素 | 维度 | 评分等级 |
|------|------|----------|
| **1. 影响** | 影响范围 | 1-5级 |
| | 影响程度 | 1-5级 |
| **2. 解决问题** | 问题复杂性 | 1-5级 |
| | 问题解决 | 1-5级 |
| **3. 领导力** | 领导范围 | 1-5级 |
| | 领导方式 | 1-5级 |
| **4. 沟通** | 内部沟通协调 | 1-5级 |
| | 外部沟通协调 | 1-5级 |
| **5. 知识经验** | 知识经验范围 | 1-5级 |
| | 知识经验级别 | 1-5级 |
| **6. 岗位性质** | 环境舒适性 | 1-5级 |
| | 工作均衡性 | 1-5级 |
| | 工作时间特征 | 1-5级 |
| | 可替代性 | 1-5级 |

### 2.2 评分标准

每个维度采用 1-5 级评分制：
- **1分** - 最低要求
- **2分** - 较低要求
- **3分** - 中等要求
- **4分** - 较高要求
- **5分** - 最高要求

### 2.3 分数计算

- **岗位总分** = 14个维度分数之和（满分70分）
- **岗位平均分** = 所有评估人对该岗位评分的平均值
- **排名依据** = 岗位平均分（从高到低排序）

## 七、安全机制

### 7.1 权限分离

- **评估人**：仅能评分，无法查看其他人的评分结果
- **管理员**：需要密码验证，可查看所有数据和进行管理操作

### 7.2 数据保护

- 评分提交后自动锁定
- 解锁需管理员操作，且有日志记录
- 所有管理操作均有日志留存

### 7.3 数据校验

- Excel导入时严格校验数据格式
- 提供详细的错误报告
- 支持下载错误报告文件
